Today I learned

-history of linux
-bash command

  • $ ls : show the list of lower-level folders under top-level folder

    • $ ls-a : show the files including the hidden files
    • $ ls -l : to confirm the details of file/folders

  • $ cd: move the location

    • $cd .. , $cd ../.. : move to upper level folder

  • $ touch hello.py: create the file ‘hello.py’

  • $ mkdir dev: create the directory ‘dev’

-vim text editor

  • normal mode: command(press ESC on any mode)
  • insert mode: edit text file(press i on normal mode)
    (press :q(quit) or :wq(save and quit))
  • visual mode: do with text block(press v on normal mode)

  • start git with $ git init

  • start git with $ git clone
